Plan

The work with Uptempo starts with the planning phase. But before you can plan your actual investments, your organization must first set the goals for the next fiscal year. This is done in strategic planning, see Align planning with strategy.

In the second step, you can build on the strategic planning to plan the investments and create corresponding investment plans. This is described in the section Plan investments.
