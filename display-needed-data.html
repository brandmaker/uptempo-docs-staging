Change the type of data that is displayed

As you manage the investment plan through different phases (planning, forecasting, managing of POs and actuals etc.), you need to view different kinds of data in the Investments section. For example, in the planning phase you edit the Plan columns, while in the forecasting phase you work with the Forecast columns. You can change views to change the kind of data that is displayed according to your needs.
