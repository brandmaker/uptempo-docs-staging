Release: September 2025

The September 2025 release for Hive9 is available from September 24, 2025, and adds these features to the platform:

Added Amazon S3 bucket support to Measure Extract service

Measure Extract, Hive9's data ingestion service, now supports the Amazon S3 protocol. If you use AWS to store files, this means you can now ingest and process files into Hive9 directly from your Amazon S3 bucket.

This new capability can pull all files from a configured S3 bucket, and parse and process them in Hive9 in the same way as other sources. With the appropriate permissions, Measure Extract can also automatically files into an "Archive" folder in the same directory path on the S3 bucket after they have been successfully processed.

Automatically collapse sections in Inspection Window

To enhance the performance of the Tactic Inspection Window, all sections (except for the main details section) are now automatically collapsed (closed) when the window is opened. This ensures that the Tactic Inspection window loads quickly, and only loads other sections as and when they are needed.

To learn more about these features and changes, download the Hive9 User Guide.
