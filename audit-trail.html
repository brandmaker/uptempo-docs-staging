Uptempo Documentation Help

View a record of events on budgets and investments

The audit trail automatically records events that take place in budgets and investments, providing you with a complete historical log of who did what and when.

If any Uptempo user modifies a budget or investment plan, you can see exactly who made the change. This gives you full visibility into all user actions at both the budget and investment plan levels, ensuring clear accountability across your team.

To export the Audit Trail list open the Roll-up or details panel. The Audit Trail list will be exported as a .csv file. Please note that the export has a limit of 2000 entries.

Open the audit trail

You can access the audit trail from either the Budgets or Investments section. In both cases, the data you see is context-specific, and the audit trail lists events relevant to the area you're viewing:

  • Budgets: If you open the audit trail from...

    • A folder or sub-folder: All events for all budget folders and sub-folders in the Uptempo instance.

    • An investment plan: Only events for the selected investment plan.

  • Investments: If you open the audit trail from...

    • A category or sub-category: All events for the category or sub-category, and for all sub-categories, line items, and placeholders within it.

    • A line item or placeholder: Only events for the selected line item or placeholder.

View the audit trail for a budget folder, sub-folder, or investment plan

  1. In the Budgets section, find the folder, sub-folder, or investment plan for which you want to view the audit trail.

  2. Click Open Roll-Up Panel (in the Panel column) on the folder, sub-folder, or investment plan.

  3. In the Roll-Up Panel, click Audit Trail.

The Audit Trail dialog opens, and displays a list of events scoped to the relevant area.

    View the audit trail for an investment item

    1. In the Investments section, find the investment item (category, sub-category, line item, or placeholder) for which you want to view the audit trail.

    2. Click Open Panel beside the name of the investment item to open its details panel.

    3. In the investment's details panel, click Audit Trail.

    The Audit Trail dialog opens, and displays a list of events scoped to the relevant area.

      Navigate and filter the audit trail

      Audit trail table columns

      The Audit Trail dialog displays the list of events in a table with these columns:

      Time

      The timestamp of the date and time the event took place.

      User

      The email address of the user associated with the event (such as the user who performed the action).

      Investments/Folder (Budgets only)

      The folder/sub-folder or investment plan that the event relates to. This column is only shown when viewing the audit trail from the Budgets section.

      Area

      The category of the event, such as the type of action or object that the event relates to.

      Details

      A short description of the event and the specific resources involved.

      Click any column header to sort the table by that column. Click the header once to sort in descending order, and click it a second time to sort in ascending order.

      Filters

      To make it easier to find the event records you need, you can use the controls above the table to filter the data that it displays. The following filters are available:

      Filter name

      Description

      To use

      Time

      Only show events that took place within a specific date range

      Choose between preset options (such as Last 1 Day, Last 30 Days, etc.), or manually specify a date range by selecting a From and To date

      User

      Only show events related to a specified user

      Enter all or part of an email address to show all events where the keywords appear in the User column

      Area

      Only show events related to a specified event category/type

      Select the area you want to filter by from the list

      Action

      Only show events related to a specified type of action, such as created or deleted

      Select the action type you want to filter by from the list

      You can specify one or multiple of the filters. After setting your filter parameters, click Search to apply them to the table.

      Export the audit trail

      You can export the data from the audit trail. The exported data is provided in the CSV file format, which you can view with any spreadsheet program.

      Each export:

      • Can contain up to 2000 event records (lines)

      • Includes all currently displayed records based on the filters applied

      Export audit trail records as CSV

      1. Open the Audit Trail panel to the scope you want to use.

        • See Open the audit trail for instructions.

      2. Optional: Apply filters to limit the displayed records as needed. The export will include only records that match the filters applied.

      3. To export the data from the audit trail, click Export.

      The CSV file with the exported data is downloaded to your device.
