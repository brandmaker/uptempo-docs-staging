Edit activity type groups

For any existing activity type group, you can change the following settings:

  • The activity type group's name

  • The Timeline Color setting

  • Which attributes are attached, and which are set as required

Edit an existing activity type group

  1. In the Activities section, click Settings:

  2. In the Activity Configuration menu, click Activities > Types & Groups.

  3. In the list panel on the left, click on the activity type group you want to edit. The selected group's settings are shown in the settings panel on the right.

  4. Make changes to the group's settings as needed.

  5. Optional: Make changes to the activity type group's attributes on the Attached attributes tab:

    • For any attached attribute, select or deselect the Required option to change whether the attribute is mandatory.

    • Click Detach Attribute to remove an attribute from the activity type group.

    Warning

    If you detach an attribute for which values have been set (on activities of types in the group), those attribute values will be permanently lost.

  6. All changes are saved automatically, and take effect immediately.
