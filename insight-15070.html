Available from November 11, 2025.

Ratio Metrics to 2 Decimal Places

The values for the ratio metrics recently added to the Time Series card will now be displayed to 2 decimal places.

Select Group Updates

The 3-dot menu options available for each group in the Select Groups tool have been updated. We are now using 'Sub-Level' to denote the hierarchy level below the selected one (vs. 'Children'). We've also added an 'Unselect All Sub-Levels' option to the top-level 3-dot menu, enabling you to easily unselect any and all selected groups across the entire hierarchy - i.e., a blank slate option.
