Make changes to your Uptempo user profile

You can update your personal details, and adjust your user preferences, on your user profile.

View and edit your user profile

  1. On any page in Uptempo, click the User Menu (the button with your initials) in the page header.

  2. In the User Menu, click My Profile:

  3. The Edit Profile page opens, and displays your personal information and settings:

  4. To edit your first name, last name, and title, use the fields in the User Profile to make changes.

Your changes are saved automatically, and take effect immediately.
