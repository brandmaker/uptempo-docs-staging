Create import templates

If you regularly import data using the same file structure and the same settings, you can create an import template to save time and effort.

Create an import template

  1. In the Budgets section, click Import and select either POs or Actuals.

  2. On the first step of the import assistant, use the Import Location menu to select the folder, sub-folder or investment plan that you would like to import into.

    • The template will be available to all users who can import to the location you select here (as well as all subordinate folders, sub-folders, and investment plans within it).

  3. Click Browse on the File to import field, and select the file to import.

  4. Complete each of the remaining steps of the import assistant, and enter the settings you want to save in the template.

  5. In step 4 of the import assistant (Map Important Data), click Action and select Import and Save Template As....

  6. Edit the name of the template in the Create New Template dialog.

  7. Click Create to finish creating the import template and start the import.

The file is imported, and you have created an import template.
