Release: June 2025

The June 2025 release for Hive9 is available from June 25, 2025, and adds these features to the platform:

Prevent Disruptions from Inactive Approval Workflows

You can now safely manage approval workflows without breaking active processes, as Hive9 blocks deactivation or deletion of workflows currently in progress and provides clear guidance when editing tactics with inactive workflows. The system automatically prevents approval actions on tactics using deactivated workflows, while still showing you which workflows are assigned. So you always understand what needs to be updated before you can move forward with your changes or approvals.

Share Customized Plan and B&C Grid Views

With the latest update, users can now share their personally configured grid layouts and filters between Plan and B&C grids with other users. This collaborative feature eliminates redundant configuration work across teams, as recipients can access your shared views directly from their My Saved Views section in Plan grid, complete with all column selections, display order preferences, and filter settings intact.

Terminology Changes in UI

With shared grid views, we’ve updated terminology throughout the views functionality to make the interface more intuitive and consistent. The term Views replaces Filter views to reflect what users actually interact with. Button labels now clearly match their actions, particularly changing the confirmation button in the sharing dialog from Save to Share to better represent its function.
