Manage user access to budgets

In Uptempo Financial Management, users can only access budgets (folders, sub-folders, or investment plans) that they have been invited to. Administrators control which users can view or edit each budget, maintaining clear accountability across users in the instance.

Invite users to budgets

Before you begin

These requirements only apply to Uptempo 1.0 instances. If you are using an Uptempo 2.0 instance, you can skip this section.

Before you can invite users to a budget, you must:

  • Create at least one permissions role for the Budget module (under Administration > Permissions).

  • Add the Module Access permission to each permissions role in the Budget module.

Invite a user to a folder, sub-folder, or investment plan

  1. In the Budgets section, find the budget (folder, sub-folder, or investment plan) that you want to invite a user to.

  2. On the selected budget, click invite under the Manage Users column. The Share: dialog opens to the Invite tab:

  3. On the Invite tab, enter the user's email address, first name, and last name.

    The user receives the invitation and any subsequent password reset emails at this address. Make sure the email address you enter is the user's primary email address.

  4. Use the Assign User Role menu to select a user role for the selected level of the hierarchy.

    • The selected user role determines the invited user's access level and permissions.

    • To learn more about the available user roles, see User roles.

  5. Optional: If you're inviting a user to a folder or sub-folder, they are also invited to all of its sub-folders and investment plans by default. To limit their access to only the selected folder or sub-folder, set the Folders setting to Invite to this folder only.

  6. Optional: Use the Invite Subject and Invite Message fields to customize the invitation email that is sent to the user.

    Add an action and a deadline to the message to encourage users to register and log in. For example: "Please review the Q2 Targets before [DATE]."

  7. Under Notification, select the Send email to invited user option.

    If you leave this option deselected, the user receives access to the specified budget, but does not receive an email notification. This is useful if you're giving access to additional budgets to an existing user, or if you're inviting a user to multiple separate budgets, but don't want them to receive a separate email notification for each one.

  8. Optional: To send yourself a copy of the invitation email, select the Send a copy to myself option under Notification.

  9. To finish inviting the user to the selected budget, click Invite.

The invited user receives the specified access level to the selected budget with immediate effect.

If you chose to notify the user by email, they are also sent an email with a link to register and sign in to Uptempo Financial Management.

The link in the invitation email expires after 72 hours.

    Resend user invitation emails

    If an invited user did not receive the invitation email, or if the link in their invitation email expired before they could use it, you can resend their invitation email at any time.

    Resend an invitation email to a user

    1. In the Budgets section, find the budget (folder, sub-folder, or investment plan) that you invited the user to.

    2. On the selected budget, click edit under the Manage Users column. The Share: dialog opens to the Shared With tab.

    3. In the table, select the user to whom you want to resend the invitation email.

      The Status column shows the current status of the user's Financial Management account, including whether an invitation was sent.

      The status also reflects the level of the budget hierarchy the user was invited to. For example, if a user was invited to a folder containing investment plans, but was not invited to the individual investment plans within it, their status on the folder would show Invitation Sent, but their status on the investment plans would show Invitation Not Sent.

    4. Click Send Invite.

    5. In the confirmation dialog, click Yes to finish resending the invitation email.

    The invitation email is sent. If the user had not previously been sent an invitation email, their status is also changed to Invitation Sent.

      Remove users from budgets

      You can remove users from individual budget folders, sub-folders, or investment plans at any time. Removing a user from a given budget does not affect their access to other parts of the budget hierarchy that they are invited to.

      Remove a user from a budget

      1. In the Budgets section, find the folder, sub-folder, or investment plan you want to remove the user from.

        If a user must be removed from the entire hierarchy and was invited at multiple levels, remove them at each level they were invited to.

      2. In the Manage Users column, click Edit on the folder, sub-folder, or investment plan. The Share: dialog for the selected budget opens to the Shared With tab.

        • If you are removing a user from a folder or sub-folder, choose whether the changes apply to all sub-folders and investment plans within it, or to this folder only.

        • Users with the Owner role can't be removed from any folder, sub-folder, or investment pan.

      3. In the table on the Shared With tab, select the user you want to remove.

      4. Click Remove.

      The user is removed from the selected folder, sub-folder, or investment plan.
