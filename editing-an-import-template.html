Edit import templates

If you need to make a change to an existing import template, you can edit it.

Edit an import template

  1. In the Budgets section, click Import and select either POs or Actuals.

  2. On the first step of the import assistant, use the Import Location menu to select the folder, sub-folder or investment plan that you would like to import into.

  3. Click Browse on the File to import field, and select the file to import.

  4. Use the Choose import template menu to select the template you want to edit.

  5. Navigate through each of the steps of the import assistant and edit the settings you want to change the template.

  6. In step 4 of the import assistant (Map Important Data), click Action and select Import and Save Template to save your changes and start the import.

The file is imported, and the import template is updated with the edited settings.
