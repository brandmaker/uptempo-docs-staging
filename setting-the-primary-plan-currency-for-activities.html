Set the primary plan currency

You can set the primary currency for activities according to your preferences. All monetary amounts on all activities (such as Estimated Costs) for all activities are shown in the primary currency you select.

If an activity is connected to an Uptempo Financial Management investment, and the investment is in a different currency than the primary plan currency, the funds amounts are converted to the primary plan currency. The exchange rates used for the conversion are managed in Uptempo Financial Management under Budgets > Master Settings > Currency.

Set the primary plan currency for the activity hierarchy

  1. In the Activities section, click Settings:

  2. In the Activity Configuration menu, click Currencies.

  3. On the Primary Plan Currency page, use the Currency menu to select the currency you want to use:

  4. The change is saved automatically, and takes effect immediately.
