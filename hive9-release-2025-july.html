Release: July 2025

The July 2025 release for Hive9 is available from July 30, 2025, and adds these features to the platform:

Custom notifications for non-date attribute

You can receive notifications when the status of your tactics, campaigns, or programs changes. Track approvals, ownership transfers, and workflow updates without manually checking dates. Select the attribute changes that matter to you and how you want to be alerted: in-app, email, or both.

Web interface for uploading financial transactions to Hive9

You can now upload financial transactions directly in Hive9. Simply click the new Upload Transactions button, download our template for the correct format, and import your transaction data. This feature requires the Upload Financial Transactions via UI permission to be enabled for your role.

Support for formula-based custom attributes at the PR Transactions level

You can now use formulas in custom attributes to automate calculations and dynamically populate fields within PR Transactions. When you create or edit a custom attribute for PR Transactions, you can now use formulas in the same way as you can in other hierarchy levels (for example Plan, Campaign, Program, Tactic, Line Items). Formula-based values in custom attribute fields are calculated whenever a PR Transaction is created or updated, and are displayed as read-only in the POR Transaction form and reports. For help with configuring formulas at the PR Transaction level, please reach out to your Customer Success Manager.

To learn more about these features and changes and how to use them, download the Hive9 User Guide.
