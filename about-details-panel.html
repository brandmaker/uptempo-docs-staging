About the investment details panel

The investment details panel in the Investments section displays data about each investment item (category, sub-category, line item, or placeholder). The details panel is divided into sections, which each show different types of data. These include:

Details

Displays basic data about the investment, and lets you view and set attribute values for the investment.

Metrics

Contains different metrics configured for your investment plans, for example your top-down targets, and variance between plan and actual spend, allowing you to view your metrics while planning your bottom up spend.

Strategy

Displays objectives defined with Strategic Planner.

Alignment

Contains information about your alignment score and provides calculations as you make updates to your investment plans while ensuring your plan aligns with your strategic objectives.

Activities

Allows you to connect activities from Uptempo Campaign Management to the investment, and manage how the investment's funds are allocated to the connected activities.

The sections in the details panel vary depending on the features that are enabled in your Uptempo instance.

Open an investment's details panel

  1. Go to the Investments section.

  2. In the Hierarchy column, click Open Panel on the investment category, sub-category, or line item whose details you want to view.

The selected investment's details panel is displayed.

    Uptempo ID and Persistent ID

    Uptempo ID

    Marketing investment and spend data are often tracked across multiple systems such as a CRM, procurement system, finance ERP, event tracking platform, digital marketing platforms, and many others. With the data dispersed across the systems it can be challenging to accurately connect all data points for comprehensive reporting.

    Uptempo provides marketers with a solution to align their data across all systems with a single unique identifier. Uptempo Financial Management automatically generates unique IDs for each budget folder, sub-folder, and investment plan, and each investment category, sub-category, placeholder, and line item when it’s created. The unique ID for each item can be found on the Roll-Up and details panels, and is named according to the type of item, for example Line Item ID or Investments ID.

    An item's ID can be used at three levels.

    • At the budget hierarchy level, it can be connected to other planning tools.

    • At the investment category or sub-category level, it can be tagged to POs that contain multiple lines or campaigns that have many programs or a parent campaign with many children.

    • At the investment line item level, it can be tagged to individual PO line items, spend or expense transactions, campaigns, projects and/or investments where appropriate.

    Persistent ID

    In addition to the Uptempo ID, Financial Management also generates a Persistent ID for each category, sub-category, or line item. This ID allows for one consistent reference ID to be used across multiple years. This way, only one ID will need to be used across the systems year over year.

    When line items are copied from one year to the next, during roll-over, a Persistent ID is created which indicates that there was an original version of this object in a previous year. If you copy an item, the original item's Persistent ID is retained on the copy, while a new Uptempo ID is also created (which is specific to the budget year).

    The Persistent ID links back to the original item, and exposes all related items. The items that share a Persistent ID do not impact each other: if a value changes in one item, it does not affect any other item with the same Persistent ID. If a new item is created for the first time in a hierarchy without being copied from a previous year, the Persistent ID will be the same as the Uptempo ID.

    Persistent IDs are not generated for budget level items like folders, sub-folders, and investment plans.

    An item's Persistent ID can be changed, but only through the API. Reach out to your Customer Success Manager for more details.
