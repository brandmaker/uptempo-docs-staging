Release: November 2025

The November 2025 release for Hive9 is available from November 26, 2025, and adds these features to the platform:

Accrual Requests now recoverable from the Recycle Bin

The Recycle Bin in Hive9 now also supports Accrual Requests, making it possible to undo accidental deletions.

Previously, deleting an Accrual Request (AR) was permanent. Now, whenever you delete an AR it will appear in the Recycle Bin, just like other recoverable resources. In the Recycle Bin, select the new Accrual Requests option to view deleted ARs, and click Restore to recover any deleted AR.

Support for sending Approval Workflow error notifications to non-admin users

By default, notifications about errors in an Approval Workflow are only sent to users with the Client Admin role. Because users in this role aren't always the best person to diagnose and fix problems in Approval Workflows, we've added the option to send notifications about Approval Workflow errors to users in non-admin roles.

Now, you can turn on the new role permissions setting Approval Workflows > Receive Approval Workflow Error Notification for any role: when this setting is turned on for a role, users with the role receive notifications when errors occur in Approval Workflows. With this new feature, you can ensure Approval Workflow problems are resolved quickly by alerting the right people when they occur.

To learn more about these features and changes, download the Hive9 User Guide.
