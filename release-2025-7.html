Announcement: Updates to the Uptempo feature release lifecycle

We are making some changes to our feature release lifecycle. These changes are designed to improve transparency, and to help our customers plan and prepare for upcoming product changes more effectively.

In brief, we are:

  • Renaming the Early Access stage to Beta to more clearly communicate the in-development nature of these features.

  • Adding a new Preview stage to give you time to evaluate new Uptempo features before they are automatically rolled out to all customers.

  • Publishing the details of our full feature release lifecycle (from initial availability to discontinuation), so you know exactly what to expect at each stage.

Here's what these changes mean for you:

  • Major new Uptempo Campaign Management & Financial Management features will be released in Preview first: Features in Preview are available on an opt-in basis. This means you can choose exactly when you are ready to start using them.

    • Minor enhancements (such as UX/UI and performance improvements) will continue to be released directly to General Availability, and will not be released into the Preview stage first.

  • Preview features are complete and ready to use: Unlike Beta (formerly Early Access) features, Preview features are not actively evolving, and come with full support and documentation.

  • Features in Preview automatically transition to General Availability: Preview features are made generally available after 2 monthly release cycles. This means that new features are automatically turned on for all customers approximately 2 months after they are first released in Preview.

    • For example, a feature released to Preview in the May release will become generally available on the release date for the July release (typically the last Wednesday of the month).

Initially, we are releasing only certain features to Preview. All other features will continue to be released directly to General Availability. At a later date (to be announced), we will transition to releasing all major new Campaign Management & Financial Management features to Preview first.

Just like Beta features, we'll announce Preview features in the release notes with a special tag. Going forward, we'll also announce in the release notes each month which Preview features have moved into General Availability with the release.

Campaign Management: Activity Configuration

Support for conditionally required dependent attributes

Attribute dependencies now support required dependent attributes. Before, you could not use an attribute as a dependent attribute if it was required on any activity type. This update removes that restriction, so that you can now make any dependent attribute required. In dependency chains of multiple linked attributes, this means you can make dependent attributes conditionally required: they're required only after certain values are selected in the controlling attribute. Together with this release's other improvements to attribute dependencies, this change makes it possible to build dynamic, streamlined forms in the activity Details panel that significantly reduce the time needed to create an activity.

Learn more about these attribute dependency enhancements

Campaign Management: Activities

User experience enhancements for attribute dependencies

We've made several UX improvements in the details panel to make it easier and faster to work with attribute dependencies, and to support required dependent attributes:

  • Grouped attribute controls for dependencies: If an activity type includes one or more attribute dependencies, the details panel now groups all attributes that are part of a dependency into their own section, and visually shows how the attributes are linked to one another. Within this section, the system does not auto-save attribute values as soon as they are selected. Instead, values are only saved when you click Save: this helps to protect data integrity by ensuring that all required dependent attributes have a value selected before changes are saved in the system.

  • Dynamically hidden dependent attribute fields: In chained dependencies that contain multiple linked attributes, the system now automatically hides all dependent attributes until they're needed. For example, if a conditionally required dependent attribute is only valid for certain values in the controlling attribute, it is initially not shown in the details panel: it only becomes visible after you select one of the controlling attribute values that make it required. This simplifies the details panel, and makes it easier to use for activity types with a lot of attributes.

  • Auto-select dependent attribute values: The system now saves you unnecessary clicks when a dependent attribute only has one possible value. Whenever you choose an option in a controlling attribute that leaves only one valid option in its dependent attribute, the system simply selects that option for you — no extra click needed.

Learn more about these attribute dependency enhancements

Financial Management: Investments

Connect multiple activities to a line item in a single step

Connecting multiple Campaign Management activities to a Financial Management investment line item is now faster and easier. You no longer need to connect activities to a line item one-by-one: in the redesigned activity picker, you can select multiple activities at once, then connect them all to the line item with a single click.

Learn more about connecting activities and investments

Improved filtering capabilities for tracking activity funding

We have added more filtering options to the investment hierarchy table to give you better insight into activity funding and funds allocation. Before, you could filter the table to see only investments that are connected to activities. With the new and expanded Filter by Activity Funding options, you can now also see the specific amounts allocated to connected activities. And the opposite view is now also available, so you can see investment amounts that are not yet allocated to any activity. These new filter options provide a more complete view of how your investments support activities, helping you to manage budgets and make informed decisions.
