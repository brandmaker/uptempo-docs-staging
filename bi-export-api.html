BI Export API

Overview

The BI Export API gives you programmatic access to a set of raw data files that represent your budgeting data. This data includes all columns and rows from all budgets as well as the mapped purchase orders, invoices, and external campaigns.

The BI Export API consists of a set of API endpoints that return individual datasets in the form of a CSV file. There is also an endpoint that retrieves a ZIP file containing CVS files of all available datasets. The BI Export API documentation describes the endpoints and the data contained in the returned datasets.

The BI Export API is now deprecated. This legacy API will be retired with a future release.

All data available through the BI Export API is also available from the Allocadia API. If you are currently using the BI Export API, please migrate to the Allocadia API as soon as possible to ensure continued support.

For more information, see the Allocadia API documentation.
