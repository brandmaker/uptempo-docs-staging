Release: October 2025

The October 2025 release for Hive9 is available from October 29, 2025, and adds these features to the platform:

Support for monthly currency conversion rates on Purchase Requests

Previously, Purchase Requests in Hive9 could only use a single annual foreign exchange (FX) conversion rate for each currency. To enable more granular and time-sensitive conversions that better reflect real-world accounting practices, Hive9 now supports setting monthly FX rates for Purchase Requests.

Monthly FX rates were already supported for Actuals, and you can now apply the same rates to Purchase Requests: under Settings > Currency, there is now a new Use Monthly Rates for Purchase Requests setting for each currency. As with Actuals, enabling this option will apply the conversion rates set in the Monthly Exchange Rates fields to Purchase Requests in the specified currency (if this option is not enabled, the currency will use the set annual conversion rate, as before).

Customize the default attribute to search on Hive9 grids

As an administrator, you can now choose which attribute Hive9's grid search will use by default. Until now, the search function on the Plan, Budget & Cost, and Calendar grids defaulted to searching by the Name attribute, and users had to manually change the attribute to search. If your organization typically uses an attribute other than Name to search for records in Hive9 (such as ID), you can now use the Default Search Attribute for Grids setting to set that attribute as the default for your users.

Highlighted changes in Purchase Request update Excel files

When a Purchase Request (PR) is updated, the cells with changed values are now highlighted in the Excel file attached to the PR update email. This enhancement makes it easier for reviewers to identify what has changed compared to previous versions, helping to save time and reduce errors.

To learn more about these features and changes, download the Hive9 User Guide.
