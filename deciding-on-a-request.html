Approve or deny investment approval requests

After an investment has been submitted for approval, the designated approvers must review the approval request.

As an approver, you can view open approval requests in the Budgets section, and decide whether to approve or deny them.

Review and decide on approval requests

  1. In the Budgets section, find the investment plan for which you want to decide on an approval request.

  2. In the Approvals column, click approve to open the Approvals page, which displays the queue of pending approval requests as a table.

    if there are requests to be reviewed, the approve text is displayed in red, and the number beside it indicates the number of current pending approval requests.

  3. Optional: To view more details about a request, click Open Panel on the request under the Line Item column to open its details panel.

  4. To make a decision about a pending request, click on its row in the table to select the request, then use the buttons above the table to select your action. You can choose between the following actions:

Approve

Approves the selected request. If needed, use the Comment field to add any comments, then click Approve to finish approving the request.

Deny

Rejects the selected request. If needed, use the Comment field to add any comments, then click Deny to finish rejecting the request.

Send Back

Sends the request back to the submitter for revision. Use this option if an investment needs changes before approval. If needed, use the Comment field to add any comments, then click Send back to finish sending the request back to the submitter.

  1. Optional: Repeat steps 4-6 to make decisions about other pending requests.

  2. When you are finished, click Go Back to exit the Approvals page.
