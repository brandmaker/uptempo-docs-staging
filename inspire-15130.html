v15.13.0

Available from April 29, 2026.

Indexed Percent of Comp

Recent improvements were made to how Percent of Comp is presented in the Inspire yield curve view. The baseline has shifted from 0 to 100 so values now read like an index: 100 equals comparison, above 100 is lift, below 100 is under-spend. In both the aggregated table and individual yield curve cards, labels now use 'Agg Pct of Comp' and 'Pct of Comp,' and the editable Spend input ranges from 0-250 (0-2.5x comparison).

Card Value Drift Indicator

A yellow indicator appears when card-level values drift more than 1% from the aggregate, with a one-click option to sync all cards.

CSV Export Updates

The CSV export now uses matching "Spend Pct of Comp" and "[KPI] Pct of Comp" headers, and exported values reflect the same 100-based baseline as the UI.
