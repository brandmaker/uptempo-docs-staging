Delete actuals

If needed, you can delete all of the actuals created by an import, or just a single actual.

Owners, Administrators, Editors and Editor-Data Entry Only users can delete actuals that they have imported. Only Owners and Administrators can delete actuals that other users have imported.

Delete imported actuals

Delete all actuals created by an import

  1. In the Budgets section, click Import History and Schedules in line with the investment plan from which you want to delete actuals.

  2. In the Import History and Schedules dialog, scroll to the rightmost column in the Past Imports tab. Click Delete in line with the import file that created the actuals you want to delete.

  3. Click Yes to delete the imported actuals.

You have deleted the actual import.

    Delete an individual imported actual

    1. In the Budgets section, click Import History and Schedules in line with the investment plan from which you want to delete actuals.

    2. In the Import History and Schedules dialog, find the import file that created the actual you want to delete in the Past Imports tab. Click on the number in the Mapped or Unmapped column (depending on where the item is) to view individual actuals created by the import.

    3. Scroll to the rightmost column and click Delete in line with the actual you want to delete.

    You have deleted an individual imported actual.

      Delete manually created actuals

      Delete a manually created actual

      1. In the Investments section, click Open Panel in line with the investment for which you would like to delete a actual.

      2. On the details panel, click Actuals to open the Actuals dialog.

      3. In the Actuals dialog, click Delete in line with the actual you want to delete.

      You have deleted a manually created actual.
