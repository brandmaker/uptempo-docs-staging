Deprecations and Discontinuations: 2025.4

From time to time, we must remove functionality from the Uptempo platform for various reasons. To give users time to prepare and adjust, we communicate functionality removals in two stages:

  1. Deprecation: Deprecated functionality is still available in the Uptempo platform as of the current release, but is planned to be removed with a future release. Deprecated functionality usually does not receive any further updates, and only remains available to already existing users.

  2. Discontinuation: Discontinued functionality has been removed from the Uptempo platform as of the current release, and is no longer available.

If functionality that you use is announced as deprecated on this page, we recommend that you begin preparing for its removal as soon as possible to ensure a smooth transition.

Deprecations

With this release, the following functionality has been deprecated:

Basic Authentication for BI Export API

As of the 2025.4 release, we are deprecating basic authentication as a method for accessing the Uptempo/Allocadia BI Export API. This change affects users of all supported versions of the BI Export API.

Previously, basic authentication was the only supported authentication method for legacy APIs, including the BI Export API. To enhance security for users of these APIs, we have incorporated the BI Export API endpoints into the Allocadia API, which uses bearer token authentication. As a result, you can now interact with the BI Export API endpoints via the Allocadia API, allowing you to access the same capabilities in a more secure way. For more information about the Allocadia API, including how to authenticate, see our API documentation.

Because the Allocadia API now supports all the functions of the legacy BI Export API, accessing the BI Export API using basic authentication is now deprecated, and will be discontinued in the future. As with all Uptempo product deprecations, the basic authentication method will remain available until further notice, and no discontinuation date has been set at this time. However, if you currently use the BI Export API, we strongly encourage you to migrate to the Allocadia API at your earliest convenience to ensure a smooth transition, and to benefit from modern, secure authentication for your requests.

Discontinuations

With this release, the following functionality has been discontinued:

Enable Tasks and Worklogs Setting

The Enable Tasks and Worklogs setting has been removed from Other Settings in Datasheet Engine Administration. This setting was obsolete, as the associated product functionality had been discontinued previously.

This change simplifies the interface by removing an unused setting. No action is required from you, and this removal won't affect any existing configurations.
