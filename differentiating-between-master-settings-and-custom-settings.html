Financial Management configuration overview

Uptempo Financial Management has two levels of settings that let you configure features globally, or for specific parts of your budget hierarchy: master settings and custom settings.

We are actively migrating the Financial Management documentation. If a topic isn't available here yet, you can still find it on the legacy documentation site.

Master settings

Master settings are the global settings where you turn on and configure features across the entire budget hierarchy. Changes to master settings affect all investments in the hierarchy.

In some cases, you may want to configure different features for a specific sub-folder. For example:

  • If sub-folders are organized by region, you can add specific columns and components for regions that track different Key Performance Indicators (KPIs).

  • Administrators can be assigned at different levels of the hierarchy, so different regions only see the financial data they have access to.

Custom settings

Custom settings appear at the sub-folder level and affect only the budgets in that sub-folder. Some master settings options aren't available at the custom settings level:

  • Currency: Currency settings are managed in master settings only. You can set custom currencies for each investment plan there, so you don't have to update each sub-folder individually.

  • Strategic Planning: Options for the entire budget hierarchy are set in master settings to keep configuration centralized.

  • Workflow: All investments in the budget hierarchy appear in the master settings workflow tabs.

  • Connectors: Connectors are only accessible from master settings because they require data exchange across the entire hierarchy.

  • Configuration: Preferences are set in master settings to centralize configuration across the hierarchy.

The following options can be adjusted at the custom settings level:

  • Columns: Add columns and fields for specific budgets on the Investment grid, Actuals panel, Details panel, Grand Total panel, PO panel, and Roll-up panel.

  • Dependencies: Create dependencies between two Drop-Down List or Multi-Select List fields on the Details panel to help with data entry for specific workspaces.

  • Page Layout: Create and adjust what appears in the Investments section, including views, metrics, and columns.

  • Categories: Define categories for specific investment plans so they're already populated in the investment grid.
