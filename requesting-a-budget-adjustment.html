Request a budget adjustment

If you need more budget for a plan that is performing well, you can request a budget adjustment.

Request a budget adjustment

  1. In the Investments section, select the investment plan from which you want transfer budget.

  2. Click Request Transfer above the grid. The Request a transfer dialog is opened.

  3. Enter the amount that should be adjusted.

    • If you want to reduce the budget, enter a negative amount, for example -10,000.00.

  4. Make sure you have selected the correct currency.

  5. In the Type field, select Adjustment.

  6. In the To field, select the investment plan whose budget is to be adjusted.

  7. In the field right to the To field, select the applicable budget attribute.

  8. Optional: Enter a description in the field at the bottom of the dialog.

  9. Click Send.

You have request a budget adjustment.
