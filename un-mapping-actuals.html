Unmap actuals

If you discover that an actual was assigned to the wrong line item during the import, you can easily correct it. Simply unmap the actual first, and then manually map it to the correct line item.

When a record is unmapped, any changes that had been made to the record will be lost.

Unmap an actual

  1. In the Investments section, locate the investment with the actual that needs to be unmapped and right-click on it.

  2. Select Go to Actuals.

  3. Select the actual you want to unmap and scroll to the far right.

  4. Click Unmap.

You have unmapped the actual. The unmapped record will move to Map column in the Budgets section. Now you can locate, view and remap the record from the Map column.
