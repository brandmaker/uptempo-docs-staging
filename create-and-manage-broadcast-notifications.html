Broadcast in-app messages to your organization

As an administrator, you can use Broadcast Notifications to display short messages to all users in your organization. By default, these messages are automatically displayed to each user the next time they sign in to Uptempo:

Broadcast Notifications are useful for communicating important or time-sensitive information about your Uptempo instance. For example, you can use them to keep your marketing teams informed about changes in your Uptempo instance, or to communicate key dates and deadlines.

Create and manage Broadcast Notifications

Before you begin

  • This functionality is available in Uptempo 2.0 instances for all packages.

    If you're not sure which version or package of Uptempo you're using, or if you're interested in upgrading, please get in touch with your Customer Success contact.

  • To configure this functionality, you must have administrator access to your Uptempo instance.

Create Broadcast Notification messages

You can create one Broadcast Notification message at a time. If you create a new message while there is already an existing message, the active message will be removed and replaced with the new message.

Broadcast Notifications are visible to all users in an Uptempo instance, and can't be restricted based on user role or access permissions.

Create and publish a Broadcast Notification message

  1. On any page in Uptempo, click the Broadcast Notification button in the page header to open the Broadcast Notifications menu:

  2. Click Create new to open the Create Broadcast Message dialog:

  3. Enter your message into the Message Body field:

    A message can be up to 255 characters long. While you're writing a message, Create Broadcast Message dialog shows a counter of the number of characters you've used so far.

  4. Optional: By default, the message is automatically displayed to all users the next time they sign in. To change this behavior, deselect the Pop-up for all users option:

    When this option is turned off, a message notification badge is displayed on the Broadcast Notification button in the page header:

    To view the message, users must click the Broadcast Notification button.

  5. To save and display your new message, click Publish.

Your Broadcast Notification message is published, and is displayed to all users in your organization.

If you selected the Pop-up for all users option, the dialog with the message will only be automatically displayed to each user once. After a user dismisses the message dialog, it will not be automatically shown to them again until a new message is published.

If needed, users can reopen the currently active message at any time by clicking on Broadcast Notifications > Current message:

    Edit Broadcast Notification messages

    After you have published a Broadcast Notification message, you can make changes to it at any time.

    Edit a published Broadcast Notification message

    1. On any page in Uptempo, click the Broadcast Notification button in the page header to open the Broadcast Notifications menu.

    2. Click Edit to open the Edit Broadcast Message dialog.

    3. Use the Message Body text field to make changes to the message.

      Even if the Pop-up for all users checkbox is selected in this dialog, the message will not be automatically displayed again for any users who had already seen and dismissed the previous version.

      If you are editing the message to add important information that should be seen by all users, publish a new message instead.

    4. To save your changes and publish the edited message, click Save.

    Users who view the message will now see the edited version.

      Remove Broadcast Notification messages

      You can remove a published Broadcast Notification message at any time to stop displaying it.

      When you remove a message, it is deleted permanently. A removed message's content can't be recovered.

      Remove a published Broadcast Notification message

      1. On any page in Uptempo, click the Broadcast Notification button in the page header to open the Broadcast Notifications menu.

      2. Click Remove.

      3. To stop displaying the published message, click Remove in the confirmation dialog.

      The Broadcast Notification is removed with immediate effect, and is no longer visible or accessible to any user in your Uptempo instance.
