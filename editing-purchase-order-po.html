Edit purchase orders (POs)

If you need to make changes to an existing purchase order, you can edit it.

Edit a PO

  1. In the Investments section, click Open Panel in line with the investment that is mapped to the PO you want to edit.

  2. On the details panel, click PO to open the PO dialog.

  3. In the PO dialog, click the PO to be edited and click the Edit Purchase Order button.

  4. Optional: If the PO you want to edit is split, click Manage Split. The dialog Manage Split for PO is displayed. The list shows the source investment plan, source category and source line item for every split.

    • To add another split: 1. Click Add Location. Another table row is displayed. 2. For the added row, select investment plan, category and line item in the respective columns. 3. Repeat steps 1 and 2 for every split you need for this PO. 4. Enter the percentage for each split. Make sure your split percentages total exactly to 100%.

    • To delete a split:

      1. Click Delete in line with the split you want to delete. The table row is deleted.

      2. Repeat the previous step for every split you want to delete.

      3. Edit the percentage for each remaining split. Make sure your split percentages total exactly to 100%.

      4. Click Done.

  5. If needed, edit PO Number.

  6. If needed, edit PO Amount.

  7. If needed, edit amortization of the PO (for more information see Amortizing: One PO, multiple months):

    • If no amortization is needed, select Single Date, and enter the accrual date in PO Start Date. Proceed with step 8.

    • If amortization is needed, select Date Range, and enter the first accrual date in PO Start Date and the last accrual date in PO End Date.

      1. Click Manage Amortization. The Manage Amortization for PO dialog is displayed.

      2. Optional: If you are amortizing a split PO, select PO split in Manage for dropdown. The PO amount for the split is displayed next to Amount Remaining.

      3. Select:

        • If the remaining amount is amortized equally during the amortization period: Click Amortize Equally.

        • If you need to amortize unequally: Enter the respective monthly amounts in the fields under Amount Remaining.

      4. Optional: If you are amortizing a split PO, repeat steps 2 and 3 for every split.

      5. If your amortization period extends into the next fiscal year, enter the rollover amount in the appropriate field. This amount will be transferred accordingly when creating the new fiscal year.

      6. Click Save.

  8. If applicable, edit other data like PO Vendor or PO Description.

  9. Click Save.

You have manually edited a PO.
