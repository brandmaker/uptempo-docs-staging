Delete attributes

If you no longer need an attribute, you can delete it at any time.

Deleting an attribute permanently removes the attribute and all of its data. If the attribute has values on any activities, those values will be permanently deleted as well.

After deletion, the attribute and its values cannot be restored.

If you no longer want to use an attribute, consider removing it from all attribute types where it is used instead. This will hide the attribute from activities but will retain its associated data.

Delete an attribute

  1. In the Activities section, click Settings:

  2. On the Activity Configuration page, the Attribute Definitions section should be displayed by default.

    • If you're on a different page, click Attributes > Attribute Definitions in the navigation menu.

  3. The table in the Attribute Definitions section displays a list of all existing attributes. Find the attribute you want to delete and click on it to select it and open its Attribute Detailspanel.

    • You can also enter a keyword into the Search field to filter the table so that only attributes with a name or type matching the keyword are shown.

  4. Use the Attribute Details panel to check that you have selected the correct attribute.

  5. Click Delete.

  6. In the Confirm Delete dialog, click Yes to delete the attribute.

The attribute is removed from the Attribute Definitions list, and from all activity types and activities where it was used, with immediate effect.
