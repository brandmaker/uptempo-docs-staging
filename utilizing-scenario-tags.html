Tag line items with scenario tags

You can tag line items with scenario tags to help you adjust your budgets and prioritize investments in case circumstances change. For example, you could tag expenses as options if additional budget becomes available, or tag other expenses as candidates for cuts if you need to reduce budget.

About scenario tags

All untagged line items are considered the planned investments against the original budget target. Scenario tags can be used for "nice-to-have" items if there was extra budget and for new events/campaigns if additional funds became available. Uptempo has the following default scenario tags:

  • 15% Increase (blue) – Ideally what we could execute if there was a 15% increase in allocation.

  • 10% Increase (green) – What we want to execute in the event of a 10% increase in allocation.

  • Cancelled (yellow) – This investment will no longer be executed

Best practices

  • Tags can be customized and may vary between organizations. A best practice are the tags Wishlist, Out of Scope, and Cancelled.

  • Don't delete investments with scenarios even if the scenario is not realized. This can help provide additional context and serve as a reference point for next year’s scenario planning.

Tag line items with scenario tags

Apply a scenario tag to a line item

  1. In the Investments section, place the pointer on a line item to highlight it.

  2. Beside the line item name, click Set scenario to open the scenario tag menu.

  3. Select the scenario tag you want to set on the line item.
