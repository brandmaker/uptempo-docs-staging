Enter an amount in a different currency

If a spend amount is incurred in another country, you may need to enter it in a currency that's different from the investment plan currency. For example, your investment plan may be set to use USD, but an invoice from a supplier in Europe is denominated in EUR.

In this situation, you can use the set a multi-currency value option. With this option, you enter the amount in the original currency, and Uptempo automatically converts it (at the current exchange rate) and records the amount in the investment plan currency.

Set a multi-currency value

  1. In the Investments section, open the investment plan where you want to enter the spend amount.

  2. On the row of the applicable line item or placeholder, right-click on the input cell where you want to enter the amount to open the context menu.

    The context menu won't appear if the cell is currently in edit mode (if you left-clicked on the cell first).

  3. In the context menu, click Set Multi-currency Value.

  4. In the dialog, use the Base Value field to enter the amount in the original currency, and the Base Currency menu to select the original currency.

  5. Verify the conversion rate and converted amount (Calculated Value).

  6. To save the value to the cell, click Save.

The value is saved, and displayed in the investment plan currency.

When you enter a multi-currency value, Uptempo saves the original currency and amount. To display the original amounts and currencies in the investment grid, select the display option Show Original Currency.
