Release: May 2025

The May 2025 release for Hive9 is available from May 28, 2025, and adds these features to the platform:

Finance Budget Link for Multi-Year Tactics Mandatory

You must now link finance budgets to all line-items in each year of your multi-year tactics. When creating or editing line-items in any year in a single tactic spanning multiple years, ensure every fiscal year has a budget link. Otherwise, you will see a validation message. This message will require you to add the missing links before saving. This applies to all Hive9 dialogs and windows where you manage line items.

This validation is applicable only if the corresponding setting is turned ON at the “Org. Setup” screen.

Save Column Order in Grids

You will no longer lose your carefully arranged column layouts when making grid modifications. Previously, adding or removing columns would reset your entire column arrangement. With this version, your previously set personal column sequence is retained for all changes to plan, B&C, PR and AR grids.

Reset Selected Pre-Saved Filters in Grids

The Set Default View feature lets you reset your default view with a single click. Your default view can be a Saved Preset or another filtered/unfiltered view which you have not saved as a preset. You find “Set Default View” in the My Saved Views dropdown in Plan, B&C, PR, and AR grids. This reset applies only to your current session. To make this state persist across sessions, you must make sure the desired Preset has been marked as Default.

Asana Integration

This connector was released in March 2025. It is only available to a select group of customers. It allows you syncing tactics as nested tasks under parent projects in Asana while providing customizable configurations for pushing tasks based on tactic type. Template sync options have been relocated to the Integration Instance screen with improved logging for better visibility.

Terminology Changes in UI

We have updated the terminology across grids to make the interface more intuitive and consistent. The term Views replaces Filter Views to reflect what you actually interact with. Button labels now clearly match their actions.
