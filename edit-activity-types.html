Edit activity types

For any existing activity type, you can change the following settings:

  • The activity type's name

  • Which activity type group the type is assigned to

  • The Timeline Color setting (if the type is not part of an activity type group)

  • Which attributes are attached, and which are set as required

  • The activity type's options, such as:

  • Assigned KPIs

  • Configured Outbound Actions

  • Whether optional features like Estimated Costs or Impact Modeling are enabled (or set as required)

Edit an activity type

  1. In the Activities section, click Settings:

  2. In the Activity Configuration menu, click Activities > Types & Groups.

  3. In the list panel on the left, click the activity type you want to edit. The selected activity type's settings are shown in the settings panel on the right.

  4. Make changes to the activity type's settings as needed.

    The following settings can only be edited if the activity type has no existing activities:

    • The assigned activity type group

    • The linked workflow (however, you can change which workflow type variables are assigned to synchronize with attached attributes)

    • The sync direction for synchronized attributes

    Additionally, the Estimated Costs and Connect to Spend features can only be turned off if there are no existing activities of the type that have data related to these features.

  5. Optional: Make changes to the activity type group's attributes on the Attached attributes tab.

    • For any attached attribute, select or deselect the Required option to change whether the attribute is mandatory.

    • Click Detach Attribute to remove an attribute from the activity type group.

    Warning

    If you detach an attribute for which values have been set (on activities of this type), those attribute values will be permanently lost.

  6. All changes are saved automatically, and take effect immediately.
