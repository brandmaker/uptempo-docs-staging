Copy investments

If some investments in your plan are used repeatedly, you can save time by copying and pasting them instead of creating them individually.

Options for copying investment items

When you copy an investment item, you can choose between different options to define the scope of what will be copied.

The following options are available for categories and sub-categories:

Category/Sub-category Structure only (Includes Line Items)

Copies the selected category/sub-category row and the rows of all sub-categories and line items nested within it. Does not copy attribute values or cell values.

Category/Sub-category Structure and Attributes

Copies the selected category/sub-category row and the rows of all sub-categories and line items nested within it. Also copies attribute values, but does not copy cell values.

Category/Sub-category Structure, Attributes and Cell Values

Copies the selected category/sub-category row and the rows of all sub-categories and line items nested within it. Also copies all attribute values and cell values.

The following options are available for line items and placeholders:

Line Item/Placeholder Only

Copies only the selected line item/placeholder row. Does not copy attribute values or cell values.

Line Item/Placeholder and Attributes

Copies the selected line item/placeholder row. Also copies attribute values, but does not copy cell values.

Line Item/Placeholder, Attributes and Cell Values

Copies the selected line item/placeholder row. Also copies all attribute values and cell values.

Create copies of an investment

Copy an investment item

You can create copies of any category, sub-category, line item, or placeholder. Copies of categories and sub-categories also include all sub-categories and line items within them. Copies can only be made within the same investment plan: you can't copy investment items between two different investment plans.

  1. Go to the Investments section and open the investment plan in which you want to copy an investment item.

  2. Right-click the investment item you want to copy.

  3. In the menu, place the pointer on Copy Row to view options for copying the selected investment. Select the option you want to use for this copy.

  4. Right-click the investment item where you want to create the copy. The copy will be created as a child of this investment item.

    • If you copy a category onto another category or sub-category, the copy will be created as a sub-category instead.

    • To create a copy at the top level of the investment hierarchy, select the Grand Total row as the destination for the copy.

    • To create a copy of a line item or placeholder alongside the original, select the category or sub-category where the original is located as the destination for the copy.

  5. In the menu, select Paste Row to create the copy in the selected location.

The copy is created in the selected location of the investment hierarchy, with the name "Copy of [original investment item name]".
