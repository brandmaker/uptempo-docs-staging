Transfer plan amounts to forecast columns

After your organization has completed the planning phase, you can start to execute on the investments and maintain the expected costs in the Forecast columns. In order to start with the values from the plan columns, you can exports the plan data and import it into the Forecast columns. This saves you the manual work of transferring it.

Transfer data from plan to forecast columns

  1. In the investment plan where you want to transfer data from plan to forecast columns, click the Export options button on the right above the Investment grid and select Export All fields for Import. An XLS file is created that contains all data of the investment plan.

  2. Open the XLS file and scroll right to the Plan columns (Jan Plan, Feb Plan etc).

  3. Copy all data from the Plan columns (except headers entries) and paste it to the applicable Forecast columns. Make sure that you copy the amounts correctly:

    • Amounts must be pasted at the same line item or placeholder, that means in the same row.

    • Amounts must be pasted in the Forecast column of the same month.

    We recommend not to change any of the column names in the first row.

  4. Save the file.

  5. In the Budgets section, click Import and select Investment. The Import Data dialog is opened.

  6. In the Import to * field, select the investment plan to which you want to import the data.

  7. In the File field, select the XLS file in which you copied the data from Plan to Forecast columns.The file is uploaded.

  8. Click Map Fields. A list of all Uptempo fields and File fields is displayed. Since you have not changed any of the column names, the automatic mapping should work without errors. To be sure, enable Show unmapped items only. If unmapped items are displayed, map them.

  9. Click Merge Data .The data is imported and copied into the fields. It is possible that warnings are displayed because plan columns or cells are locked.

  10. Click Done.

  11. Open the investment plan.

  12. Check the Forecast columns to make sure that the data has been transferred correctly.

You have transferred the data from the Plan to the Forecast columns.
