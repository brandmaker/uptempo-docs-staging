Undo an action

If you make a mistake while working wiht the investment grid (such as entering an incorrect value or creating a duplicate entry), you can use the Undo action to quickly correct it.

The Undo history is cleared once you leave the Investments section, or when you refresh your browser tab.

Undo actions on the investent grid

  1. In the Investments section, click Undo above the investment grid.

  2. The last action is reversed.

  • You can't undo a deletion, or undo the same type of action more than once. For example, after creating 3 line items, the Undo button will only allow you to undo the last line item that was created.

  • There is no limit to the amount of actions you can undo, but there is also no "redo" function.

  • The Undo function does not apply to actions in any investment's details panel, or to copying or moving investments.
