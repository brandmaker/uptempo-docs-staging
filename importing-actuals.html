Import actuals

After you have received and accepted your ordered services and the corresponding invoices have been processed, you can import the actuals into the system. This allows you to easily compare your forecast status with actual spending without needing to enter the data manually. See Create actuals by importing.

The import process will automatically map most actuals to their corresponding line items. However, some actuals might remain unmapped due to issues like missing or incorrect identifiers. When this happens, you must manually map the unassociated actuals to the correct line item. See Map imported actuals.

If you discover that an actual was assigned to the wrong line item during the import, you can easily correct it. Simply unmap the actual first, and then manually map it to the correct line item. See Unmap actuals.
