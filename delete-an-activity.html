Delete activities

If you no longer need an activity, for example because it was created by mistake, you can delete it.

Deleting an activity will remove the activity itself and its data, which includes:

  • Attribute data

  • Estimated costs

  • Connections to investments (and any existing funds allocations)

  • Impact and performance data

Warning

Deleting an activity permanently removes all data associated with the activity.

Deleted activities or their data can't be recovered.

Delete an activity

  1. In the Activities section, find the activity you want to delete in the activity hierarchy panel, and click on the activity's name. The activity's details panel opens on the right:

  2. On any tab of the details panel, hold the pointer on Actions and click Delete.

If the activity you want to delete still has child activities, you must delete or move the child activities before you can delete the activity.

  1. In the confirmation dialog, check the box that you understand the consequences of deleting the activity, then click Delete Activity.

The deleted activity is immediately removed from the activity hierarchy.
