Create activities

An activity is any planned initiative you create to promote a product, service, or message to a defined audience. Activities are the core building blocks of Campaign Management and represent every level of your marketing hierarchy, from high-level plans and campaigns down to individual tactics. Each activity is configured, scheduled, tracked, and measured so you can understand its contribution to marketing impact.

For more information, see About activities and the activity hierarchy.

Create new activities

Create an activity

You can add new activities to the activity hierarchy at any time.

  1. In Uptempo, click Activities in the navigation sidebar.

  2. In the Activities section, click Create Activity. The button is available on both the Timeline and Summary display modes:

    The Create Activity setup assistant opens with the Type page displayed.

  3. On the Type page:

    1. Use the Activity type list to select the type of activity you want to create from the available options. You can also type a keyword into the field to search for matching options.

    2. Use the Parent list to select the activity under which you want to create this new activity in the hierarchy. You can also type a keyword into the field to search for matching options.

    Depending on the rules set up in your Uptempo environment, selecting a parent activity may be optional for some activity types.

  4. Click Next. The Details page is displayed.

  5. On the Details page:

    1. Enter a name for your new activity into the Name field.

    2. Optional: To set the date range when this activity will be in-market, use the In-market Dates fields to choose a start and end date.

    3. Depending on the activity type you selected, additional attribute fields may be displayed. Complete any other attributes as needed. If an attribute is marked with an asterisk (*), it is required, and you must select or enter a value before you can proceed.

    The icons Controlling Attribute and Dependent Attribute indicate that an attribute is part of a dependency. Hold your pointer on an icon to see the linked attribute's name:

    The options available in the dependent attribute are filtered by the value of the controlling attribute. If no value is selected in the controlling attribute, no options are available.

  6. Click Next. The Budget page is displayed.

  7. On the Budget page:

    1. Optional: To enter a cost estimate for this activity, use the Estimated Costs field. For instructions, see Track and analyze activity budgets.

    2. Optional: To connect this activity to an investment, click Add funding source. For instructions, see Connect investments to track activity funding.

  8. Click Next. The Impact page is displayed.

  9. On the Impact page:

    • Optional: If the parent activity you selected for this activity allows child activities with impact data (or if the activity you're creating does not have a parent), you can configure the impact data here. For instructions, see Track and analyze pipeline impact.

  10. Click Submit to finish creating the activity.

The Create Activity panel closes, and the new activity's details panel opens automatically.

Need to make changes on an earlier step in the activity setup assistant? To go directly to any step in the process, click on its name.

You can move between the different steps of the activity setup assistant at any time. The system will automatically save anything you've already entered on each step.

    To streamline the activity setup process, you can also create a new activity directly under any existing activity. When you do this:

    • The Parent field is automatically set to the appropriate parent activity (the activity under which you created the activity), so you don't need to set it manually.

    • The Activity type field is filtered to only show valid options for the selected parent activity, making it easier to choose the correct activity type.

    Create an activity as a child of an existing activity

    There are two ways to create an activity directly as a child of another activity:

    • Method 1: In the Timeline or Summary display modes, hold the pointer on the activity under which you want to create the new activity. Click Add under.

    • Method 2: Click on the activity under which you want to create the new activity to open its details panel. Hold the pointer on Actions and click Add Under.

    With both methods, the Create Activity setup assistant opens, with the Parent field preselected.

      Use Uptempo AI suggestions to create activities faster

      This section describes a feature that is currently in Preview, and is available on an opt-in basis. To learn more, see What's new: 2026.6.

      All Uptempo AI features require your explicit consent before activation, even after they reach general availability.

      Contact your Customer Success Manager if you would like to use Uptempo AI features in your Uptempo instance.

      When you create a new activity, Uptempo AI helps to speed up the process and reduce manual effort by analyzing the activity to suggests values for its attributes.

      Use Uptempo AI suggestions when creating a new activity

      Currently, Uptempo AI can suggest values for single-select (dropdown) attributes. Support for additional attribute types is planned for upcoming releases.

      1. Start creating a new activity.

      2. Select the activity type and its parent activity (if needed), then click Next.

      3. On the Details step, the AI Suggestions indicator shows the status Active to indicate that the system is standing by to generate suggestions:

      4. To generate suggestions, enter a name for the activity. Uptempo AI will start, and the AI Suggestions indicator shows Generating...

      5. As soon as suggestions are available the status changes to Ready, and Uptempo AI displays the available suggestions beside each attribute:

        Depending on Uptempo AI's confidence level for an attribute, you'll see:

        • A single suggestion if Uptempo AI has high confidence that it's correct.

        • Up to three suggestions for medium-confidence attributes.

        • No suggestions, if none meet the minimum confidence threshold.

        In addition, Uptempo AI will not display suggestions for single-select attributes that have more than 50 list options.

      6. Click a suggestion to set it as the attribute's value.

      In all cases where Uptempo AI makes a suggestion, you can still select a different value manually if you want (including after you accept a suggestion, if you change your mind).
