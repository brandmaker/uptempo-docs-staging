Manage user access to budgets (Preview)

This document describes a feature that is currently in Preview, and is available on an opt-in basis. To learn more, see What's new: 2026.5.

If you have not enabled this Preview feature in your Uptempo instance, please refer to Manage user access to budgets instead.

In Uptempo Financial Management, users can only access budgets (folders, sub-folders, or investment plans) that they have been invited to. Administrators control which users can view or edit each budget, maintaining clear accountability across users in the instance.

Invite users to budgets

You can invite users to any budget folder, sub-folder, or investment plan either individually (all Uptempo versions), or as a group (Uptempo 1.0 only).

When you invite users, you also assign them a user role, which determines the permissions that they have within Uptempo Financial Management.

Before you begin

These requirements only apply to Uptempo 1.0 instances. If you are using an Uptempo 2.0 instance, you can skip this section.

Before you can invite users to a budget, you must:

  • Create at least one permissions role for the Budget module (under Administration > Permissions).

  • Add the Module Access permission to each permissions role in the Budget module. Without this permission, invited users will not be able to access the Budgets or Investments sections in your Uptempo instance.

Invite users to a budget individually

You can invite users to a budget individually, using their email address as an identifier:

  • If an email address does not exist in the Uptempo instance yet, inviting it creates a new user.

  • If an email address is already registered to an existing user in the instance, inviting it grants additional access to that user (for example, to another folder or investment plan).

When you invite a user, you can also choose to send them an email notification, optionally with a customizable message.

If you have set up Single Sign-On (SSO) for your Uptempo instance, the email notification does not include the registration link, as the user does not need to set a password for their Uptempo account.

Invite an individual user to a budget

  1. In the Budgets section, find the budget (folder, sub-folder, or investment plan) that you want to invite a user to.

  2. On the selected budget, click invite under the Manage Users column. The Share: dialog opens to the Invite tab.

  3. On the Invite tab, select Email in the Invite by menu.

  4. Enter the user's email address into the Email field.

    The user receives the invitation and any subsequent password reset emails at this address. Make sure the email address you enter is the user's primary email address.

  5. If the email address is not yet registered to an existing user in the Uptempo instance, enter the user's first and last name.

    • If a user with the email address already exists, the First Name and Last Name fields are not shown.

    • If the existing user already has access to this budget, a notification is displayed beside the Email field. You can enter another email address to invite a different user, or click cancel to close the Share: dialog.

  6. Use the Assign User Role menu to select a user role for the selected budget.

    • The selected user role determines the invited user's access level and permissions.

    • To learn more about the available user roles, see User roles.

  7. Optional: If you're inviting a user to a folder or sub-folder, they are also invited to all of its sub-folders and investment plans by default. To limit their access to only the selected folder or sub-folder, set the Folders setting to Invite to this folder only.

    • This setting is not displayed if you selected an investment plan.

  8. Optional: Use the Invite Subject and Invite Message fields to customize the invitation email that is sent to the user.

    Add an action and a deadline to the message to encourage users to register and log in. For example: "Please review the Q2 Targets before [DATE]."

  9. Under Notification, select the Send email to invited user option.

    If you leave this option deselected, the user receives access to the specified budget, but does not receive an email notification. This is useful if you're giving access to additional budgets to an existing user, or if you're inviting a user to multiple separate budgets, but don't want them to receive a separate email notification for each one.

  10. Optional: To send yourself a copy of the invitation email, select the Send a copy to myself option under Notification.

  11. To finish inviting the user to the selected budget, click Invite.

The invited user receives the specified access level to the selected budget with immediate effect, and the user is added to the table on the Shared With tab of the Share: dialog.

    If you selected the Send email to invited user option, the user is sent the following email:

    Subject: Your Uptempo login Hi, [Admin Name] ([admin email address]) has invited you to access one or more Investments they created in Uptempo. Please continue with registration using the following link: [Registration URL] Once registered, you can access the system anytime at our secure login page: Login at: [Login URL] Email: [User email address] Password: As specified during registration ---- Thanks, The Uptempo Team Contact Us: http://support.uptempo.io About Uptempo: Uptempo Marketing Performance Management helps CMOs and marketing teams manage their plans and investments, while giving full visibility into results and ROI. To learn more, visit https://www.uptempo.io

    If you specified a custom message in the Invite Message field, your custom message is added to the email body just before the signature, under the following heading:

    Message from [Admin Name] ([admin email address]):

    When the invited user clicks the registration link in the email, they are taken to the registration form, where they can enter their name and set their password:

    The link in the invitation email expires after 72 hours.

    If an individually invited user did not receive the invitation email, or if the link in their invitation email expired before they could use it, you can resend their invitation email at any time.

    Resend an invitation email to a user

    1. In the Budgets section, find the budget (folder, sub-folder, or investment plan) that you invited the user to.

    2. On the selected budget, click edit under the Manage Users column. The Share: dialog opens to the Shared With tab.

    3. Select Individuals to view the table of users individually invited to the selected budget.

    4. In the table, select the user to whom you want to resend the invitation email.

      The Status column shows the current status of the user's Financial Management account, including whether an invitation was sent.

      The status also reflects the level of the budget hierarchy the user was invited to. For example, if a user was invited to a folder containing investment plans, but was not invited to the individual investment plans within it, their status on the folder would show Invitation Sent, but their status on the investment plans would show Invitation Not Sent.

    5. Click Send Invite.

    6. In the confirmation dialog, click Yes to finish resending the invitation email.

    The invitation email is sent. If the user had not previously been sent an invitation email, their status is also changed to Invitation Sent.

      Invite users to budgets as a group

      Instead of inviting users to budgets individually, you can invite a permissions role. When you invite a permissions role, all users assigned to that role (under Administration > Permissions > Budget) are invited and granted the specified access together.

      When you invite a permissions role to a budget, you can add or remove individual users to or from the permissions role to control their access to the budget:

      • Adding a user to a permissions role also grants that user access to all budgets the permissions role is invited to

      • Removing a user from a permissions role also removes that user's access to all budgets the permissions role is invited to (unless the user was also individually invited to a budget in addition to their access through the permissions role)

      Inviting users to Financial Management budgets by their permissions role is useful if you manage your Uptempo user accounts in an external Single Sign-On Identity Provider (IdP).

      In this case, you can save time and effort by assigning users or groups in the IdP to permissions roles, then giving them access to Financial Management by inviting them collectively via their permissions role (rather than individually).

      Invite a permissions role to a budget

      1. In the Budgets section, find the budget (folder, sub-folder, or investment plan) that you want to invite the users to.

      2. On the selected budget, click invite under the Manage Users column. The Share: dialog opens to the Invite tab.

      3. On the Invite tab, select Permissions Role in the Invite by menu:

      4. Use the Permissions Role menu to select the permissions role that contains the users you want to invite.

        • All users with the permissions role receive access to the budget.

      5. Use the Assign User Role menu to select a user role for the selected budget.

        • All users with the invited permissions role receive the permissions to the budget specified by the assigned user role.

        • To learn more about the available user roles, see User roles.

      6. Optional: If you're inviting a permissions group to a folder or sub-folder, all of its sub-folders and investment plans are also automatically included in the invitation. To limit access to only the selected folder or sub-folder, set the Folders setting to Invite to this folder only.

      7. To finish inviting the permissions role to the selected budget, click Invite.

      All users in the invited permissions role receive the specified access to the selected budget with immediate effect. The permissions role is added to the table on the Shared With tab of the Share: dialog.

      Uptempo doesn't send an automated email notification to users invited to a budget by their permissions role. If you are giving access to new users, make sure to notify them separately that they now have access to Uptempo Financial Management.

        Remove users from budgets

        You can remove users from specific budget folders, sub-folders, or investment plans at any time. Removing a user from a given budget does not affect their access to other parts of the budget hierarchy that they are invited to.

        Remove an individually invited user from a budget

        If you invited a user to a budget individually (using their email address), you can also remove them individually.

        Remove an individual user from a budget

        1. In the Budgets section, find the budget (folder, sub-folder, or investment plan) that you want to remove the user from.

          If you need to remove a user from the entire hierarchy and they were invited to multiple separate folders, sub-folders, or investment plans, you must remove them at each level they were invited to.

        2. On the selected budget, click edit under the Manage Users column. The Share: dialog opens to the Shared With tab.

        3. Select Individuals to view the table of users individually invited to the selected budget.

        4. In the table, select the user you want to remove from the budget. The selected user's row in the table is highlighted.

          Users with the Owner role can't be removed from any folder, sub-folder, or investment plan.

        5. Optional: If you're removing a user from a folder or sub-folder, they are also removed from all of its sub-folders and investment plans by default. To remove them from only the selected folder or sub-folder, set the Folders setting to Apply changes to this folder only.

          • This setting is not displayed if you selected an investment plan.

        6. Click Remove:

        7. To finish removing the selected user from the budget, click Yes in the confirmation dialog.

        The user is removed from the selected folder, sub-folder, or investment plan with immediate effect.

          Remove an invited permissions role from a budget

          If you invited a permissions role to a budget, you can remove the permissions role from the budget at any time. This also removes access to the budget from all users who currently have the permissions role.

          If you need to remove access to a budget for only some of the users who were invited as part of a permissions role, you can remove them from the permissions role instead by editing their account permissions (under Administration > User List).

          Remove a permissions role from a budget

          1. In the Budgets section, find the budget (folder, sub-folder, or investment plan) that you want to remove the permissions role from.

            If you need to remove a user from the entire hierarchy and they were invited to multiple separate folders, sub-folders, or investment plans, you must remove them at each level they were invited to.

          2. On the selected budget, click edit under the Manage Users column. The Share: dialog opens to the Shared With tab.

          3. Select Permissions Roles to view the table of permissions roles invited to the selected budget.

          4. In the table, select the permissions role you want to remove from the budget. The selected permissions role's row in the table is highlighted.

          5. Optional: If you're removing a permissions role from a folder or sub-folder, all of its sub-folders and investment plans are also automatically included. To remove the permissions role from only the selected folder or sub-folder, set the Folders setting to Apply changes to this folder only.

            • This setting is not displayed if you selected an investment plan.

          6. Click Remove:

          7. To finish removing the selected permissions role from the budget, click Yes in the confirmation dialog.

          The permissions role is removed from the selected folder, sub-folder, or investment plan with immediate effect.
