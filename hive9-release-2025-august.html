Release: August 2025

The August 2025 release for Hive9 is available from August 27, 2025, and adds these features to the platform:

Support for importing custom dimensions and metrics from Google Analytics 4

Hive9’s Google Analytics 4 (GA4) data ingestion feature has been enhanced to support custom dimensions and metrics. This update gives you greater flexibility in aligning GA4 data imports with your unique reporting needs, enabling more tailored analysis and deeper insights within Hive9.

Budget Configuration settings changes now tracked in Audit Log

All changes made on the Organization Setup > Budget Configuration settings page in Hive9 are now recorded in the Audit Log, allowing administrators to easily understand any changes made to financial workflows or budgeting logic. As with other tracked events, you can view all events related to Budget Configuration settings changes in the standard Hive9 Audit Log interface. Each event captures:

  • Timestamp: The time and date the change was made

  • User: Details of the user who made the change

  • Value Details: The previous settings value before the change, and the new value after the change

  • Field Name: The name of the field affected by the change

  • Entity or Module: Specified as "Organization Settings - Budget Configuration" for all changes to the Budget Configuration settings page.

    Tracking these changes in the Audit Log provides transparency, ensures accountability and compliance, and helps with troubleshooting issues that might arise out of configuration changes.

Section label changes on Organization Setup page

To more clearly communicate how to use the "Hive9 Hierarchy Naming" and "Custom Goals" settings on the Organization Setup page, we have updated the settings labels in these sections. The revised labels are designed to be more user-friendly and easier for new users to understand. The following labels have been changed:

Hive9 Hierarchy Naming section:

  • Old Label: Plan → New Label: Level 1

  • Old Label: Campaign → New Label: Level 2

  • Old Label: Program → New Label: Level 3

  • Old Label: Tactic → New Label: Level 4

  • Old Label: Line Item → New Label: Level 5

Custom Goals section:

  • Old Label: CustomGoalTypeCode → New Label: Goal Code

  • Old Label: CustomGoalTitle → New Label: Goal Name

  • Old Label: CustomGoalDataType → New Label: Goal Type

  • Old Label: CustomGoalDescription → New Label: Description

To learn more about these features and changes and how to use them, download the Hive9 User Guide.
