Bug Fixes: 2025.6

To ensure that our users can benefit from bug fixes as soon as possible, we release bug fix updates to the Uptempo platform frequently.

The following bug fixes have been released over the past four weeks, during the period of May 28 to June 20, 2025:

API

Documentation

  • In the API documentation, the navigation sidebar is no longer cut off, and instead scrolls to display all items.

Campaign Management

Activities

  • When a user creates a new activity and also has permission to create a child activity under the newly created activity, they no longer need to refresh the page before the option to create a new child activity appears.

  • When a quick search is applied to the Activity Hierarchy, the sort order of the displayed activities is no longer "random", and is instead always alphabetical.

  • When grouping by any attribute in the Summary display mode, the Planned ROI column no longer fails to sum values (for example, in the Grand total row.)

  • On activities with chained attribute dependencies, when you make a change to a dependent attribute and then, before saving that change, make another change to a non-dependent attribute that saves automatically, clicking Cancel on the dependent attribute change will no longer revert the non-dependent attribute to its previous value.

  • The Outbound Action dialog is no longer significantly smaller in Campaign Management than its counterpart in Financial Management.

  • When you set a filter in the Activity Hierarchy, the filter no longer remains active after you edit the URL to remove the filter's parameter.

  • After duplicating an activity, trying to change Estimated Costs quarters on one of the activities no longer causes an error.

  • When setting a date range for Estimated Costs, setting an end date that is earlier than the start date no longer causes an error. Instead, the system will automatically switch the two dates in this situation.

  • When changing start and end dates on Estimated Costs, the changes no longer fail to save in some cases.

  • In the Budget Picker (for connecting investments to an activity), the Open (arrow) button on the investment selection menu is no longer unresponsive, and clicking it now opens the menu as expected.

Financial Management

Audit Trail

  • When you enter a partial username into the search field, the Audit Trail no longer fails to return entries that match the partial username.

Investments

  • When you use the search field to filter the activities shown in the Activity Picker (for connecting activities to an investment), selecting an activity to connect no longer causes the filter to be reset.

  • In the Filter by Activity Funding menu, the Open (arrow) button is no longer unresponsive, and clicking it now opens the menu as expected.

  • When an investment Sub-Category is a child of a Category or Sub-Category that is linked to an activity, it is no longer displayed without an icon that indicates the link.

  • When there are approved transfers in the hierarchy, the system now displays a message when a user adds a new currency to the hierarchy to warn users that doing so may cause unexpected conversion results for the approved transfers.
