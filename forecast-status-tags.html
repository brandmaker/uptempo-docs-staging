Use forecast status tags

As you place orders, your funds become contractually committed. Once the invoice is paid, those commitments turn into actual expenses.

Because your spending moves through these different stages, the amounts in your Forecast columns will reflect various statuses. If you ever need to reallocate funds (such as shifting a planned expense to a different month) you can use forecast status tags to clearly mark and track those changes.

Default forecast status tags

Uptempo provides the following default tags:

  • Forecast (taupe): What you think you are going to spend. Initially, this should be the same value as the planned amount but may change.

  • Committed (green): You are contractually obligated to this amount.

  • Occurred (blue): This spend has occurred during this month and should be recognized in this month's spend.

Tags may be different in your Uptempo instance: your organization may use different names, statuses or additional tags. Contact your Uptempo administrator if you have questions.

Set forecast status tags on investment amounts

Before you begin

  • To view and set forecast status tags, you must select a view that displays Forecast column in the investment grid.

  • To tag an investment with a forecast status tag, there must be an amount in the Forecast column for that investment.

Set a forecast status tag on an amount

  1. In the Investments section, click the colored dot beside the forecast amount for the investment you would like to tag.

  2. Select your desired forecast status tag from the drop-down menu.

You have tagged the investment.

    To make it easy to see which amounts are in which status, you can restrict the view of Forecast columns to specific tags.
