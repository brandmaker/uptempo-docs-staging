Map imported actuals

The import process will automatically map most actuals to their corresponding line items. However, some actuals might remain unmapped due to issues like missing or incorrect identifiers. When this happens, you must manually map the unassociated actuals to the correct line items.

Map actuals to line items

Map POs to line items

  1. In the Budgets section, select map (x) in the Actuals column in line with your desired investment plan.

    The number x in brackets of map (x) indicates how many POs need to be mapped.

  2. If the Map screen shows multiple imports to map, select the import you want to map in the Import to map drowpdown in the top left corner of the screen.

  3. Map an actual:

    • If there is no PO data available that can be used to perform an automated mapping to the associated line items:

      1. Choose the appropriate values in the Investment Plan, Category, and Line item columns to which the unmapped actuals belong.

        If there are no category groupings or line items in the investment plan, those fields will remain blank, and the actual will be mapped at the investment plan level.

      2. Click the Map (1) button.

      3. Repeat steps 1 and 2 for every record that needs to be mapped.

    • If you know that matching POs within the same level of hierarchy are available:

      1. Click Map All to PO in the upper right corner of the Map Actuals screen. The action proposes line items with matching POs. The Investment Plan, Category, and Line item columns will be filled accordingly.

      2. Check the entries in the Investment Plan, Category, and Line item columns for a record:

        • If the entries are correct, select a row and click Map (1).

        • If the entries are not correct, edit the mapping and click Map (1).

      3. Repeat the previous step until all records are mapped.

You have mapped the actuals.

    Map actuals in bulk

    You can use Bulk Map to map multiple actuals to a single line item.

    Map actuals to a line item in bulk

    1. In the Budgets section, select map (x) in the Actuals column in line with your desired investment plan.

      The number x in brackets of map (x) indicates how many POs need to be mapped.

    2. On the Map Screen, select all records that should be mapped in bulk.

      • To multi-select, hold Ctrl/Cmd and click on each of the actuals you want to select.

    3. Click Bulk Map (x).

      The number x in brackets indicates how many actuals will be mapped to a single investment.

    4. In the Bulk Map menu select the Investment Plan, Category, and Line item the actuals are associated with.

    5. Click Map at the bottom of the menu.

    All selected actuals are mapped to the selected line item.
