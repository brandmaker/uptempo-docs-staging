Edit actuals

If you need to make changes to an existing actual, you can edit it.

Edit an actual

  1. In the Investments section, click Open Panel in line with the investment that contains the actual you want to edit.

  2. On the details panel, click Actuals to open the Actuals dialog.

  3. On the Actuals panel, navigate to the actual to be edited.

  4. Edit the value in any column as needed.

  5. Repeat steps 3 and 4 for every actual to be edited.

  6. When you're finished, close the Actuals dialog.
